# FinanceIQ - Modern Financial Dashboard

A cutting-edge financial management application built with React, Vite, and Tailwind CSS. Features a sleek dark theme design inspired by modern fintech platforms, providing comprehensive financial insights and transaction management.

## 🚀 Features

### ✨ Modern UI/UX Design
- **Dark Theme First**: Professional dark interface with glassmorphism effects
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Fluid transitions and hover effects
- **Modern Typography**: Clean Inter font with proper hierarchy

### 📊 Financial Analytics
- **Real-time Dashboard**: Live financial overview with key metrics
- **Interactive Charts**: Line and pie charts powered by Recharts
- **Transaction Management**: Full CRUD operations with filtering
- **AI-Powered Insights**: Smart financial analysis and recommendations

### 🛠️ Technical Excellence
- **React 19**: Latest React features with concurrent rendering
- **Vite**: Lightning-fast development and optimized builds
- **Tailwind CSS**: Utility-first styling with custom design system
- **Context API**: Efficient state management
- **Local Storage**: Persistent data storage
- **ESLint**: Code quality and consistency

## 🎨 Design Inspiration

Inspired by modern fintech platforms like [Zorvyn](https://zorvyn.io/), featuring:
- Gradient backgrounds and glassmorphism
- Professional color schemes
- Enterprise-grade UI components
- Data-driven design patterns

## 🏗️ Architecture

```
src/
├── components/
│   ├── dashboard/          # Dashboard-specific components
│   ├── layout/            # Layout components (Sidebar, Navbar)
│   ├── transactions/      # Transaction management
│   ├── insights/          # Analytics components
│   └── common/            # Reusable components
├── pages/                 # Main application pages
├── context/               # React Context for state management
├── utils/                 # Utility functions and constants
└── App.jsx               # Main application component
```

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd finance-dashboard
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:5174
   ```

### Build for Production

```bash
npm run build
```
